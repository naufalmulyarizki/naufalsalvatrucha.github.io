Recommended


- Player Dilarang Keras Vdm Atau Menabrak Sengaja Ke Player Lain


- Player Di Larang Keras Mengunakan Bug Abuse

- Player Di larang Keras Menggunakan Karakter Invisble

- Player Di Larang Keras Menggunakan Program Ilegal/Cheat

- Player Dilarang Keras Menggunakan Script/File Tambahan ( Damage Modifier , Thermal Scope , FIrespeed , Aimbot , Ammo Modifier , Etc)

- Player Dilarang Keras Membunuh Musuh Kemudian Merevive Dan Membunuhnya Kembali Untuk Memperkaya Kill (Farming)

- Player Di Larang Berbicara Kasar Di Kolom Chat

- Player Di Larang Spawn Kendaraan Di Base Terlalu Banyak

- Player Dilarang Keluar Dari Game Ketika Sedang Berperang

- Player Dilarang Menembak Di Base dan Menembaki Orang Yang Di Base (Hijau)

- Player Dilarang Memakai Controler (Joy Stick)

- Player Dilarang Menggunakan Kill Effect Thermal Flash

- Player Dilarang Keras berbicara yang tidak baik didalam game seperti menghina orang tua, rasis, menghina agama, sara, dan mencari kebencian
